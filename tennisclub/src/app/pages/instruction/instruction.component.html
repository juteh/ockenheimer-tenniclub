Anleitung

Diese Anwendung dient dazu Mitglieder, Gäste, Getränke und Getränkelisten zu verwalten und zu exportieren.

Mitglieder können über den Button "Mitglied erstellen" erstellt werden. Es öffnet sich ein Fenster zum erstellen bzw. bearbeiten der Mitglieds. Jedes Mitglied breucht mindestens einen Vor- oder Nachnamen. Ansonsten kann er nicht erstellt werden. Nachdem drücken von "Speichern" wird das Fenster geschlossen und das Mitglied wird in der Liste angezeigt.

Wird auf das Stift-Symbol gedrückt öffnet sich ein Fenster zum bearbeiten des Mitglieds. Wird speichern gedrückt, schließt sich das Fenster und das Mitglied wird in der Liste aktualisiert dargestellt.

Wird auf das Kreuz-Symbol gedrückt wird eine Warnung angezeigt. Wird die Warnung bestätig, wird das Mitglied gelöscht und nicht mehr in der Liste angezeigt.

Gäste können über den Button "Gast erstellen" erstellt werden. Es öffnet sich ein Fenster zum erstellen bzw. bearbeiten des Gasts. Jeder Gast breucht mindestens einen Vor- oder Nachnamen. Ansonsten kann er nicht erstellt werden. Nachdem drücken von "Speichern" wird das Fenster geschlossen und der Gast wird in der Liste angezeigt.

Wird auf das Stift-Symbol gedrückt öffnet sich ein Fenster zum bearbeiten des Gasts. Wird speichern gedrückt, schließt sich das Fenster und der Gast wird in der Liste aktualisiert dargestellt.

Wird auf das Kreuz-Symbol gedrückt wird eine Warnung angezeigt. Wird die Warnung bestätig, wird der Gast gelöscht und nicht mehr in der Liste angezeigt.

Getränke können über den Button "Getränk erstellen" erstellt werden. Es öffnet sich ein Fenster zum erstellen bzw. bearbeiten des Getränks. Jedes Getränk breucht mindestens einen Namen. Liter und Preis sind initial immer 0. Ansonsten kann das Getränk nicht erstellt werden. Nachdem drücken von "Speichern" wird das Fenster geschlossen und das Getränk wird in der Liste angezeigt.

Wird auf das Stift-Symbol gedrückt öffnet sich ein Fenster zum bearbeiten des Getränks. Wird speichern gedrückt, schließt sich das Fenster und das Getränk wird in der Liste aktualisiert dargestellt.

Wird auf das Kreuz-Symbol gedrückt wird eine Warnung angezeigt. Wird die Warnung bestätig, wird das Getränk gelöscht und nicht mehr in der Liste angezeigt.

Das "Getränkeliste erstellen" führt zu einem Fenster zum erstellen der Getränkeliste. Falls eine Vorlage vorhanden ist wird diese benutzt für das Vorauswählen von Personen und Getränken. Nachdem speichern wird die neue Getränkeliste in der Liste angezeigt.

Das Stift-Symbol führt zu einem Fenster zum Bearbeiten der Werte der Getränkeliste. Nachdem speichern werden die Änderungen der Getränkeliste ind Liste angezeigt.

Das Taschenrechner-Symbol führt zu einem Fenster zum Berechnen der Getränkeliste. Es kann zu jeder Person die Anzahl der Getränke angegeben werden. Durch das Eingabefeld kann die Menge vom getrunkenem Getränk pro Person definiert werden. Es wird direkt die Summe aktualisiert. Nachdem speicher wird die Liste angezeigt mit den neuen Gesdamtkosten der bearbeiten Getränkeliste angezeigt.

Auf das klicken des Export-Symbols können Getränkelisten und Gesamtlisten exportiert werden als PDF-Dateien. Die Gesamtübersicht kann in zwei Varianten exportiert werden. Als Gesamtübersicht exportieren werden alle Personen mit ihren gesamten Ausgaben angezeigt. Bei der detailierten Übersicht werden dazu noch die einzelnen Getränke und deren Kosten angezeigt.

Dient als Vorlage von Personen und Getränke für das erstellen von Getränkelisten. Es können Personen und Getränke hinzugefügt oder entfernt werden. Nach jeder Aktion werden die Änderungen gespeichert. Wird eine Getränkeliste erstellt werden im Bereich der Personen und Getränke, je was in der Vorlage gewählt wurde, diese vorausgewählt. Die Gesamtliste kann wie die Getränkelisten bearbeitet und berechnet werden.
